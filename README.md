This repository accompanies the course *Neural Networks and Deep Learning for Scientists*, which focuses on applying modern machine learning and deep learning methods to real-world scientific problems.

## Course Overview
The course is designed for STEM graduate students and advanced undergraduates with prior programming experience. It emphasizes practical understanding over theory-heavy derivations, with a strong focus on building, training, evaluating, and interpreting neural network models.

Students learn how to use deep learning as a scientific tool, integrating data preprocessing, modeling, evaluation, and interpretation into a single workflow.

## Learning Objectives
By the end of the course, students will be able to:
- Use Python, Jupyter notebooks, and PyTorch for machine learning workflows
- Explore, visualize, and preprocess large scientific datasets
- Build and train neural networks for classification, regression, and representation learning
- Understand and apply core concepts such as overfitting, regularization, and model selection
- Implement end-to-end deep learning pipelines for scientific applications
- Critically analyze model performance, limitations, and interpretability

## Topics Covered
- Machine learning fundamentals and data pipelines
- Supervised and unsupervised learning
- Neural network anatomy and training
- Optimization, gradient descent, and backpropagation
- Fully connected networks, CNNs, and transfer learning
- Autoencoders and dimensionality reduction
- Recurrent networks, Transformers, and large language models
- Graph neural networks and advanced deep learning models
- Applications of deep learning in chemistry, biology, and related sciences

## Course Format
- Lectures introducing concepts and intuition
- Hands-on labs using PyTorch
- Programming assignments and Kaggle-style evaluations
- A team-based, open-ended final project on a scientific problem

## Assessment
- Attendance and participation
- Lab assignments
- Final open-ended project involving data, modeling, and analysis

## Tools and Platforms
- Python and PyTorch
- Jupyter Notebook / JupyterLab
- GPU computing via Google Colab and PSC Bridges-2
- Modern AI-assisted coding tools, used with proper attribution
